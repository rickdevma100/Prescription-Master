# Details:
Implementation steps:
- Create app: `npm create vite@latest doctors-prescription-frontend -- --template react-ts`
- Install deps: `cd doctors-prescription-frontend && npm i`
- ESLint/Prettier: `npm i -D eslint prettier eslint-config-prettier eslint-plugin-react eslint-plugin-react-hooks @typescript-eslint/parser @typescript-eslint/eslint-plugin`
- Configure `.eslintrc.cjs` with React 18, TS rules (strict true, noImplicitAny, exactOptionalPropertyTypes, noUnusedLocals).
- Update `tsconfig.json`: set `"strict": true`, `"jsx": "react-jsx"`.
- Create folders per PRD: `src/components`, `src/hooks`, `src/api`, `src/pages`, `src/styles`.
- Add `App.tsx`, `index.tsx` using `createRoot` from React 18.
- Add `.env.example` with `VITE_API_BASE_URL=` and `VITE_USE_MOCKS=false`.
- Pseudo-code scaffold:
